What it is #

The Imperial Codex is a smart, conversational help system for our Discord server. Ask it questions about the community, channels, bots, games, and features—and it will guide you with clear answers and links.

How to access #

  1. Mention the bot and ask for help:
    • @ImperialCodex help — open the main menu
    • @ImperialCodex commands — bot commands overview
    • @ImperialCodex music — music bot info
    • @ImperialCodex rules — server guidelines
  2. Ask in natural language:
    • “How do I play music?”
    • “Where do I get roles?”
    • “Show me game channels.”

Pro tips for users #

  1. Be specific: “help with music commands” or “how to get roles.”
  2. Use Search—it’s smart and forgiving.
  3. Check Quick Access—your shortcuts get better over time.
  4. Follow breadcrumbs to stay oriented.
  5. Try alternate phrasing if you don’t see what you need.

Getting started #

  1. Mention the bot and say “getting started.”
  2. Browse the main menu categories.
  3. Visit “About Us” to learn the basics.
  4. Open “Roles” to tailor your notifications.
  5. Explore “Commands” to see what our bots can do.

Interactive elements #

  • Context-aware help that adapts to you.
