Commands overview #

  • /embed create — create a new embed via modal
  • /embed edit — edit an existing bot embed you own (or any, if admin)
  • /embed colors — see colors available to your tier
  • /embed features — see features unlocked by your roles

Role-based access #

Features unlock progressively by tier:

Tiers

  1. Silver Fang — basic embed (title, description, color)
  2. Golden Snake — image and thumbnail
  3. Platinum Ghost — author name and icon
  4. Diamond Wraith — footer text
  5. Mystic Dragon — timestamp
  6. Moderator/Admin — full access

Available features #

  • Basic — title, description, color
  • Images — main image and thumbnail
  • Author — author name and icon
  • Footer — footer text
  • Timestamp — add created time

/embed create #

  • Purpose: open a modal with fields based on your current tier.
  • How to use:
  1. Run /embed create.
  2. Fill fields (title, description, color, etc.).
  3. Submit to post your embed.

After posting, you have a 1-hour window to self-edit your embed.

/embed edit #

  • Purpose: edit your recent embed (or any embed as admin).
  • How to use:
/embed edit <message_link>
/embed edit <channel_id> <message_id>
  • Requirements: You must be the original creator and within 1 hour of posting—or have admin permissions. Target must be a bot embed.

Important notes #

Permissions

  • Requires at least Silver Fang to use embed commands.
  • Higher roles unlock more features and colors.
  • Admins can edit any bot embed.

Time limits

  • 1-hour self-edit window after creation.
  • After 1 hour, only admins can edit.

Cooldowns

  • Commands may enforce cooldowns to prevent spam.
  • Cooldown duration varies by command and tier.

Error messages

  • ❌ You don’t have permission
  • ❌ Session has expired
  • ❌ Invalid message link
  • ❌ Not a valid bot embed

Tips for users #

  1. Plan your embed content before opening the modal.
  2. Use /embed colors to choose colors ahead of time.
  3. Check /embed features to see your options by tier.
  4. Edit within the first hour to avoid needing admin help.
  5. Grab links via “Copy Message Link” for quick editing.

FAQ #

Why can’t I see some fields in the create modal?

Fields are tier-gated. Unlock higher tiers to access advanced fields like images, author, footer, and timestamp.

Can I edit embeds older than an hour?

Only administrators can edit embeds after the 1-hour window. Ask in the appropriate channel if you need help.

My edit failed with a permissions error. Why?

You must be the original creator and within the edit window, and the message must be a bot embed. Otherwise, request admin assistance.

Create confidently. Use your tier’s tools, keep messages clear, and update within the window. 🎨
