Basics (Birthdays & Economy intro) #

🎂 Set your birthday

  • /remember-birthday <date> — Example: 1994-12-31 or 12-31
  • /forget-birthday — Removes your stored birthday

📅 See birthdays

  • /birthday — View your own or another member’s birthday
  • /next-birthday — Lists the next 10 upcoming birthdays

🔥 Economy basics

We’re actively experimenting with a custom economy system.

Start earning Embers by claiming /daily, using /work on cooldown, and unlocking achievements.

  • /buy — Purchase items from the shop
  • /items — View your inventory
  • /use — Use an owned item
  • !rc — Remove a color role before applying a new one

Server info & search tools #

🛠️ Server information

  • /role-info — Get info on any role
  • /server-info — See details about the current server
  • /user-info — Get profile info on yourself or another member

📈 Levels

  • /levels — View leaderboard link
  • /rank — Check your rank or someone else's

🔍 Search tools

  • /anime — Look up anime information
  • /imgur — Use cautiously, content varies
  • /manga — Search for manga titles
  • /pokemon — Get Pokémon info
  • /twitch — Look up Twitch streamers
  • /youtube — Search for videos & channels

Fun & games #

🎵 Music quiz (voice channel only)

  • /start-quiz — Start a music quiz (limit: 3/day)
  • /stop-quiz — End the current quiz

🎲 Quick games (any channel)

  • /dice — Roll dice to earn coins
  • /games — Get info on all reward-based games
  • /guess — Guess a number between 1–100
  • /rps — Rock, paper, scissors (earn coins!)
  • /roulette — Play Russian roulette for coins

Economy commands #

  • /buy — Purchase an item from the shop
  • /daily — Claim daily coins
  • /coins — View your or another member’s balance
  • /economy-info — Learn about how the economy works
  • /give-item — Transfer an item to someone else
  • /items — Check your inventory
  • /richest — See the top players
  • /shop — Browse the item shop
  • /use — Use an item in your inventory
  • /work — Earn coins on cooldown

Example: /buy red-color-role — gives you a red-themed name if available in shop.

Experimental feature

We’re developing a custom economy to reward a wide range of activity—chatting, leveling, playing games, and helping others. Expect frequent iteration.

  • Upcoming: player shops, auctions, achievements, item trading
  • Some features may be unstable or change often
  • Got ideas? Share feedback with our suggestion tools

FAQ #

Why didn’t my command work?

Check permissions, channel context (some commands require voice or specific channels), and cooldowns. Try with minimal options first.

How often can I use economy commands?

Commands like /daily and /work have built-in cooldowns. The bot will show remaining time if you’re early.

Where do I report bugs or request features?

Open a support ticket or post in the appropriate forum channel. Include screenshots, error text, and timestamps if possible.
