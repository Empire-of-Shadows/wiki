What it does #

Our bot helps you explore the server, track your activity, create beautiful announcement cards, share suggestions, and play community mini‑games — all with easy buttons and friendly menus.

Key features #

🎨 Profile cards

  • Your “digital ID” with level, XP, messages, voice time, streaks, badges, inventory, and Embers.
  • Switch themes and layouts with simple buttons.
  • Create your own custom color theme.

📊 User stats

  • Dashboard sections for Overview, Messages, Voice, Emojis.
  • Live refresh for up‑to‑date numbers.
  • Choose to keep it private or share a public preview.

🏛️ Imperial Codex (guide system)

  • Ask natural questions or mention the bot for help.
  • Smart search, Quick Access shortcuts, and breadcrumb navigation.
  • Covers rules, channels, roles, bots, music help, and more.

💡 Suggestion box

  • Submit ideas with optional categories or use guided templates.
  • Community voting (👍 👎 ❤️ 🤔) with live updates.
  • Clear statuses: Pending, Under Review, Approved, Implemented, Rejected, On Hold.
  • Optional DMs when your suggestion’s status changes.

🧱 Easy embeds (fancy messages)

  • Create polished embeds—titles, descriptions, colors, images, and more.
  • Role‑based access unlocks more features as you level up.
  • See which colors and features you can use anytime.

🛠️ Mod tools (for staff)

  • Clone and preview existing embeds; batch clone multiple messages.
  • Configure limits, cooldowns, allowed roles, and logging.
  • Edit any bot message embed when needed.

🔢 Mini‑game: Counting

  • Server counting channel with occasional power‑ups to keep it fresh.
  • Simple, social, and fun for everyone.

Helpful commands to get started #

Profile & stats

  • Your profile card: /member card
  • Your stats: /member stats
  • Make a theme: /theme create (see also /theme list, /theme delete)

Suggestions

  • Submit an idea: /suggest submit
  • Find ideas: /suggest search or /suggest mine

Embeds

  • Create an embed: /embed create
  • Edit an embed: /embed edit
  • See color options: /embed colors
  • See available features: /embed features

Ask the guide

  • Mention the bot and ask in plain English (e.g., @ImperialCodex help).

Why you’ll love it #

  • Express yourself with themes and stylish cards.
  • See your progress and celebrate milestones.
  • Share ideas and vote on what matters.
  • Create great‑looking announcements with ease.
  • Find answers fast with a smart, friendly guide.
  • Enjoy community games together.

Need help? #

Just mention the bot and ask in plain English. It’s built to be helpful and easy — no tech skills needed.
