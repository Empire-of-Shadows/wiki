What the bot does #

Activity tracking & rewards

  • Message activity: Earn XP and Embers as you chat.
  • Voice participation: Credit for time and activity in voice channels.
  • Daily streaks: Consistency pays—keep your streak alive.
  • Role-based rewards: Certain roles grant extra benefits automatically.

Economy system

  • Embers currency: Earn, save, and spend.
  • Inventory: Collect items and use them strategically.
  • Store: Purchase items and upgrades.
  • Trading: Exchange items with other members.

Progress tracking

  • Leveling: Gain XP to unlock features.
  • Statistics: View personal activity stats.
  • Leaderboards: See how you rank.
  • Achievements: Unlock milestones and recognition.

What information is tracked #

To power the features above, we track:

  • Message counts and activity patterns (not message content).
  • Voice channel time and activity status.
  • Reactions and emoji usage patterns.
  • Engagement trends (server interaction patterns).
  • Economy transactions (purchases, trades, item usage).
  • Achievement progress and milestones.

How your data is handled

  • Used only to provide features and rewards.
  • Stored securely using industry-standard practices.
  • Never shared with third parties.
  • Can be reset upon request.

Your privacy & control #

  • Opt-out: Options will be available.
  • Transparency: You can request your tracked data.
  • No personal content: Message content is not stored.
  • Server-scoped: Data is used only within this server.

Development status #

Early Development Notice

  • Features may change, break, or be temporarily unavailable.
  • Store and economy features are still being built and will take time.
  • Expect occasional bugs and maintenance windows.
  • Your progress and items are preserved, but experiences may be rough at times.

What’s working now #

  • Basic activity tracking.

  • Status rotation and bot responsiveness.

In active development #

  • Advanced store features.

Need help? #

  1. Check if others have similar issues.
  2. Be patient during active development.
  3. Report major bugs to server administrators.
  4. Remember: this is a work-in-progress project.

Why use this bot? #

  • Gamifies your server experience.
  • Rewards active participation.
  • Builds community engagement through shared progression.
  • Introduces a fun virtual economy to explore.
