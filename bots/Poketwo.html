Pokétwo Bot Guide

TL;DR

  • Start, pick a starter, and watch for spawns to catch quickly.
  • Manage your collection with sort, info, and select commands.
  • Battle friends, learn moves, and explore events and boxes.

Getting started #

  • @Pokétwo start — begin your adventure.
  • @Pokétwo pick <Starter Name> — choose your starter.
  • @Pokétwo help — view command list.

Core gameplay (catch & manage) #

Spawns & catching

  • @Pokétwo catch or @Pokétwo c — catch a wild Pokémon.
  • @Pokétwo hint or @Pokétwo h — get a hint on the wild spawn.

Collection & info

  • @Pokétwo pokemon — list your Pokémon with numbers.
  • @Pokétwo info — detailed info on your active Pokémon.
  • @Pokétwo pokedex — track caught/uncaught.
  • @Pokétwo order <Number|Level|IV|Pokedex> — sort your list.
  • @Pokétwo select <Pokemon Name> — set active Pokémon.
  • @Pokétwo nickname <Nickname> — rename your active Pokémon.
  • @Pokétwo release <Number> — release one; @Pokétwo releaseall to release all.
  • @Pokétwo unmega — revert a Mega Evolution.

Evolve & hunt

  • @Pokétwo evolve <Pokemon Name> — evolve if eligible.
  • @Pokétwo shiny hunt <Pokemon Name> — set your shiny target.

Battles #

  • @Pokétwo battle <@User> or @Pokétwo duel <@User> — challenge another user.
  • @Pokétwo battle add <Number> — add up to 3 Pokémon to your party.
  • @Pokétwo battle cancel — cancel the current battle.

Moves & learning

  • @Pokétwo learn <Move Name> — teach a move.
  • @Pokétwo moveinfo <Move Name> — details about a move.
  • @Pokétwo moves <Pokemon> — current and available moves.
  • @Pokétwo moveset <Pokemon> — all obtainable moves.

Progression & misc #

  • @Pokétwo daily or @Pokétwo vote — grab a mystery box via Top.gg voting.
  • @Pokétwo open <normal|great|ultra|master> [amt] — open boxes.
  • @Pokétwo event — current event info.
  • @Pokétwo embedcolor — change your current Pokémon embed color.
  • @Pokétwo next/@Pokétwo back — navigate pages while browsing.
  • @Pokétwo profile — view player profile.
  • @Pokétwo prefix <value> — set a custom prefix.
  • @Pokétwo auction <#channel> — set auction channel.
  • @Pokétwo seversilence — send level-ups via DM instead of server.
  • @Pokétwo time — check in-game time.

Tips #

  • Keep an eye on active chats—spawns trigger with activity.
  • Sort by IVs when evaluating which Pokémon to keep or release.
  • Use page navigation commands to manage long lists quickly.
  • Vote daily to build up boxes and resources over time.

FAQ #

My catch failed—what did I do wrong?

Make sure you typed the name correctly and fast enough. Use @Pokétwo hint to narrow it down.

Why can’t I see my Pokémon list?

Try @Pokétwo pokemon again and use @Pokétwo next/@Pokétwo back to page through results.

Do custom prefixes affect other bots?

No. The custom prefix you set with @Pokétwo prefix only applies to Pokétwo.
