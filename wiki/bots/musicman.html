🎵 Basic music controls #

  • /play [url/search] — add to queue
  • /pause — pause playback
  • /resume — resume
  • /skip — skip song
  • /voteskip — vote to skip
  • /volume [0–100] — change volume
  • /shuffle — shuffle queue
  • /leave — leave voice channel
  • /247 — toggle 24/7 mode

📝 Playlist controls #

  • /plhelp — playlist help
  • /plcreate [name] — create playlist
  • /pladd [name] [song] — add song to playlist
  • /plplay [name] — play playlist
  • /pldelete [name] — delete playlist
  • /plremove [name] — remove song from playlist

📻 Radio features #

  • /radio — start music radio
  • /radiolist — view radio list
  • /radiointro on/off — toggle intro

⏩ Playback tools #

  • /ff [seconds] — fast forward
  • /rw [seconds] — rewind
  • /queue — show queue
  • /qadd, /qremove, /jump, /clear
  • /history — view song history
  • /move — move bot to your VC

🎮 Fun & party features #

  • /bassboost, /pitch, /loop, /crossfade
  • /party create, /party leave, /parties

🔧 Admin commands #

  • /roleallow, /roledeny, /rolelist, /roleclear
  • /plenable, /pldisable

❓ Common issues & info #

  • Bot not responding? Try again after a few seconds.
  • Radio not working? Stop other songs first.
  • Queue bugged? Try /clear.

Use /help for a full list of features and documentation.
